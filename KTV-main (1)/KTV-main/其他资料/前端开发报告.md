## 1. 项目概述

### 1.1 产品定位

声域友是一套**线上社交KTV平台**，用户通过浏览器创建或加入虚拟房间，在房间内与朋友一起点歌、排队、实时聊天，享受线上KTV体验。

### 1.2 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                      前端应用层                              │
├─────────────────┬─────────────────┬─────────────────────────┤
│   Web用户端     │   Admin管理端   │     微信小程序           │
│   (Vue 3)       │   (Vue 3)       │     (原生开发)          │
│   Port: 5174    │   Port: 5173    │                         │
└────────┬────────┴────────┬────────┴──────────┬──────────────┘
         │                 │                   │
         └─────────────────┼───────────────────┘

                    └─────────────┘
```

## 5. 状态管理设计

### 5.1 Pinia Store架构

```
stores/
├── auth.ts        # 认证状态（token、用户信息、登录状态）
└── player.ts      # 播放器状态（队列、播放控制、同步模式）
```

### 5.2 认证状态（auth store）

```typescript
interface AuthState {
  token: string | null        // JWT令牌
  user: User | null           // 用户信息
  isLoggedIn: boolean         // 登录状态
  currentRoomId: number       // 当前房间ID（Web端）
}
```

**特性**：
- 本地持久化（localStorage）
- 跨标签页同步（storage事件监听）
- 同标签页状态同步（定时轮询 + 自定义事件）

### 5.3 播放器状态（player store）

```typescript
interface PlayerState {
  queue: PlayerTrack[]        // 播放队列
  currentIndex: number        // 当前索引
  isPlaying: boolean          // 播放状态
  currentTime: number         // 当前时间
  duration: number            // 总时长
  volume: number              // 音量
  playMode: PlayMode          // 播放模式
  syncMode: boolean           // 同步模式开关
}
```

**同步模式**：
- 服务端驱动播放状态
- 时间漂移自动校正（>2秒播放中，>0.5秒暂停）
- 拖动进度条时暂停同步

- Token自动刷新（401时自动重新登录）
