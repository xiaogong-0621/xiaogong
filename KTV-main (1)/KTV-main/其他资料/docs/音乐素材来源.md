# 音乐素材来源记录

## 素材获取方式

| 平台 | API / 方法 | 用途 |
|------|-----------|------|
| 网易云音乐 | `https://music.163.com/api/search/get` | 搜索歌曲、专辑，获取封面 URL |
| 网易云音乐 | `https://music.163.com/api/song/lyric` | 获取 LRC 歌词 |
| 网易云音乐 | `https://music.163.com/api/song/detail` | 获取歌曲详情（封面 picUrl） |
| 网易云音乐 | `https://music.163.com/api/album` | 获取专辑详情（需登录，受限） |

封面图片通过 `curl` 从网易云 CDN（`p1.music.126.net`）下载，格式为 `?param=300y300`。

## API 速查

```bash
# 搜索歌曲
curl -H "User-Agent: Mozilla/5.0" \
  "https://music.163.com/api/search/get?s=关键词&type=1&limit=5"

# 搜索专辑（type=10）
curl -H "User-Agent: Mozilla/5.0" \
  "https://music.163.com/api/search/get?s=关键词&type=10&limit=5"

# 获取歌词
curl -H "User-Agent: Mozilla/5.0" \
  "https://music.163.com/api/song/lyric?id=歌曲ID&lv=1"

# 获取歌曲详情（含封面 URL）
curl -H "User-Agent: Mozilla/5.0" \
  "https://music.163.com/api/song/detail/?id=歌曲ID&ids=[歌曲ID]"

# 下载封面（300x300）
curl -o 封面.jpg "CDN_URL?param=300y300"
```

## 注意事项

- 网易云 API 对部分请求要求手机验证（`code: -462`），专辑详情 API 较容易触发
- 网易云无周杰伦、五月天等 QQ 音乐独占版权歌曲的原版，但有翻唱版和歌词
- 封面 CDN URL 中的 hash 部分必须从 API 响应中获取，不可猜测
