## 截图要求

1. **分辨率**：建议使用1920×1080或1280×720
2. **格式**：PNG或JPG格式
3. **内容**：截图应清晰显示完整界面
4. **命名**：建议按编号命名，如"图4-1 用户注册页面.png"
5. **位置**：将截图插入到对应章节的标记位置

---

## 截图获取方法

### 管理端和Web端截图

1. 启动系统：按照项目README说明启动后端和前端服务
2. 登录系统：使用管理员账号登录管理端，使用普通账号登录Web端
3. 访问各页面：按照章节顺序访问各个功能页面
4. 截取屏幕：使用系统截图工具或浏览器开发者工具截取页面
5. 保存图片：将截图保存到docs/images/目录下

### 微信小程序截图

1. 打开微信开发者工具
2. 导入小程序项目
3. 编译运行小程序
4. 访问各页面并截取屏幕
5. 保存图片到docs/images/目录下

---

## 图片引用格式

在Word文档中插入图片时，使用以下格式：

```
图X-X 图片名称

[插入图片]
```

例如：

```
图4-1 用户注册页面

[插入用户注册页面截图]
```
